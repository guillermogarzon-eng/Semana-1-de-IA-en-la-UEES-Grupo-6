# Actividad-1.-Investigacion
Este proyecto reúne el desarrollo de los contenidos trabajados durante el laboratorio, organizados en cuatro notebooks independientes.
Cada notebook aborda una etapa diferente del análisis y procesamiento de datos, desde fundamentos de manipulación con NumPy y Pandas, visualización exploratoria, modelos básicos de Machine Learning y una introducción práctica al Deep Learning con TensorFlow/Keras.

El objetivo principal es aplicar metodologías de análisis de datos, visualización, modelado predictivo y construcción de redes neuronales, utilizando datasets de práctica y herramientas ampliamente usadas en el ecosistema de ciencia de datos.

# Cómo ejecutar el proyecto

-Ejecutar en Google Colab
-Abrir cualquier notebook desde el repositorio de GitHub.
-Seleccionar "Open in Colab" o copiar el enlace en Colab.
-Instalar dependencias si es necesario usando el archivo requirements.txt.
-Ejecutar todas las celdas en orden.
